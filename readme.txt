Display the last earnings figures from lottoland.com on your site.

== Description ==
This plugin add a widget that view the last earnings figures for a Lotterie. You can select your favorite lotterie and the figures will display in front end.

The earnings figures data is provided for free by [lottoland.com](http://lottoland.com). The widget is calling to data from the JSON API of lottoland.com to view always the last data, but caching this with the WordPress Transient API for a time window to create a better performance.

== Installation ==

= Requirements =
* WordPress 3.8+
* PHP 5.2*, newer PHP versions will work faster.

= Installation =
* Use the installer via back-end of your install or ...

1. Unpack the download-package
2. Upload the files to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it. The configuring and options are in the usual widget admin interface.
